## API

node + express + pg-mem + mongodb-memory-server


 * curl -fsSL https://deb.nodesource.com/setup_17.x | sudo -E bash -
 * sudo apt-get install -y nodejs
 * mkdir api
 * cd api
 * npm init
 * npm install express body-parser mongodb-memory-server mongodb pg-mem
